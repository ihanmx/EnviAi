## üåç About

**EnviAI** is my graduation project that addresses two modern challenges: environmental sustainability and personalized consumer experiences. This full-stack web application serves as an e-commerce platform specializing in eco-friendly products while empowering users to become designers through innovative AI and manual design tools.

### üéØ Project Goals

- **Promote Sustainability:** Encourage the use of environmentally friendly products
- **Enable Creativity:** Provide accessible design tools for product customization
- **Enhance User Experience:** Offer seamless shopping with modern web technologies
- **Leverage AI:** Integrate artificial intelligence for automated design generation
